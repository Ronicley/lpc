SELETORES

Universais: *

A principal utilização do seletor universal é "zerar" essas propriedades para todos os elementos da marcação. Como consequência o autor terá que definir explicitamente para cada elemento, na sua folha de estilos, aqueles valores. Exemlo:
*p{
Color: red;
}

SELETOR DE TIPO:

Usado para estilizar os elementos da marcação que são de um mesmo tipo; por exemplo: elemento do tipo p (parágrafo), do tipo div, do tipo ol, e assim por diante.

Usado para estilizar os elementos da marcação que são de um mesmo tipo; por exemplo: elemento do tipo p (parágrafo), do tipo div, do tipo ol, do tipo strong, e assim por diante.

SELETOR DE CLASSE:

seleciona elementos com uma classe específica aplicada: Exemplo: .nome da classe

SELETOR PSEUDO-CLASSE

Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado. Por exemplo :hover será aplicado um estilo quando o usuário passar o cursor sobre o elemento especificado pelo seletor.

Alguns exemplos

  • hover
  • link
  • visited
  • active
  • first-child
  • lang

SELETOR PSEUDO-ELEMENTO

Assim como pseudo-classes, pseudo-elementos são adicionados aos seletores, mas em vez de,
descrever um estado especial, eles permitem que você aplique estilos em certas partes de um documento.
Por exemplo, o pseudo-elemento ::first-line aplica o estilo apenas na primeira linha do elemento especificado
pele seletor.
seletor::pseudo-elemento {
propriedade: valor;
}

Exemplo:

  • ::first-line
  • ::first-letter
  • ::before
  • ::after
  • first-child
  • ::selection

CORES

Define a cor dos conteúdos textuais ou cor de frente dos elementos de marcação;
Valores podem ser :
Hexadecimal; Representação inicia-se com o sinal de tralha (#);
Palavra-chave (red);
RGB; RED(vermelho), GREEN(verde),BLUE(azul)
RGBA; Possibilidade de editar a opacidade
HSL; Tom (Hue), Saturação (Saturation), Luminosidade (Lightness);
HSLA; Possibilidade de editar a opacidade.
background: linear-gradient(hsl(0,100%,50%), hsl(60,100%,50%), hsl(120,100%,50%), hsl(180,100%,50%), hsl(240,100%,50%), hsl(300,100%,50%));

BORDAS

Define a cor dos conteúdos textuais ou cor de frente dos elementos de marcação; Decorações a serem anexadas aos quatro lados dos elementos;
  • Propriedades:

FONTES

As propriedades para estilização de fontes, definem os diferentes aspectos de apresentação dos glifos
(letras e caracteres) que compõem os conteúdos textuais dos diferentes elementos da marcação HTML.

  • color:...........cor da fonte;
  • font-family:.....família (tipo) de fontes;
  • font-size:.......tamanho da fonte;
  • font-style:......estilo da fonte;
  • font-variant:....fontes maiúsculas de menor altura;
  • font-weight:.....peso da fonte;
  • font-stretch:....grau de espansão/contração dos glifos;
  • font:............maneira abreviada para declarar todas
  • as propriedades anteriores (exceto cor).

BOXES

margin, border, padding, width, max-width, min-width, height, max-height e min-height;
  • Boxes podem ser
    1. De bloco (div, p, ul, h1, etc.);
    2. De linha (linhas de texto);

    4. Box anônimo;

DISPLAY

margin, border, padding, width, max-width, min-width, height, max-height e min-height;
  • Propriedade display determina o tipo de box para um elemento;
    1. inline:Um elemento de linha pode preencher algum texto dentro de um parágrafo <span>como esse </span> sem quebrar o fluxo daquele parágrafo. O elemento a é o elemento de linha mais comum, já que ele é utilizado para links.
    2. block: div é o exemplo de bloco mais comum. O elemento de bloco sempre começa em uma nova linha e se expande pra esquerda e direita o tanto quanto for possível. Outros elementos de bloco comuns são p e form, e agora no HTML5 temos: header, footer, section e outros.
    3. inline-block: boxes nível de bloco, dispostos como boxes inline;
    4. list-item: disposto como lista
    5. none;
Posicionamento

POSITION

O Position é uma propriedade perigosa para iniciantes. Normalmente o desenvolvedor que acaba de conhecer essa propriedade, acha que ela é a resposta para todos os problemas de posicionamento e diagramação de layout. Pelo contrário. O Position não serve para diagramar a estrutura de layouts. Para isso, você utiliza a propriedade float do css. O Position vai servir para fazer coisas mais simples. Existem 3 tipos de valores que usamos na propriedade position: relative, absolute e fixed.
